Bienvenidos al sistema de gestion de estudiantes

El Sistema de Gestión de Estudiantes es una aplicación web diseñada para facilitar el manejo de la información académica dentro de una institución educativa. Su interfaz sencilla y clara permite a los usuarios consultar los datos de los estudiantes registrados y registrar nuevos estudiantes de manera rápida y eficiente.

La plataforma está pensada para el uso del personal administrativo y académico, ofreciendo una solución práctica para centralizar y organizar los datos estudiantiles. Entre sus funciones principales se encuentran la visualización de estudiantes existentes y la incorporación de nuevos registros al sistema.

Este proyecto sirve como una base sólida para futuras ampliaciones, permitiendo integrar funcionalidades adicionales como gestión de calificaciones, materias o generación de reportes, según las necesidades de la institución.
